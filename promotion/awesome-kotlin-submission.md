## PR Title

Add Drishti SDK — accessibility infrastructure for visual STEM content

## PR Description

### What is it?

Drishti SDK is an open-source Kotlin Multiplatform library that converts visual STEM content (graphs, formulas, molecules, diagrams) into haptic feedback, spatial audio, and voice guidance for blind and visually impaired users.

### Why is it awesome?

- **Plugin-based architecture** — each content type is a separate plugin, core knows nothing about specific domains
- **Fully offline** — all ML runs on-device (no cloud calls)
- **Multiple output modalities** — haptics, spatial audio (HRTF), voice (TTS/STT)
- **Production-grade** — 665+ unit tests, full API compatibility validation, Apache 2.0 license
- **KMP** — Kotlin Multiplatform with commonMain + androidMain
